# Introduction-of-AI-and-ML-1

문일철 교수님의 '인공지능 및 기계학습 개론 I' 수업 내용을 정리하고, Christopher Bishop의 'Pattern Recognition and Machine Learning (PRML)'을 통해 수리적 기초를 심화 학습하는 저장소입니다.

---

## 학습 목표
-통계적 추정의 이해: 데이터로부터 파라미터를 추정하는 MLE와 MAP의 원리를 이해하고 수식으로 유도할 수 있습니다.

-기계학습 모델의 기초: 결정 트리(Decision Tree)와 선형 회귀(Linear Regression) 등 고전적인 기계학습 알고리즘의 최적화 과정을 파악합니다.

-확률적 분류기 마스터: 베이즈 결정 이론을 바탕으로 최적 분류기(Optimal Classifier)를 설계하고 나이브 베이즈의 조건부 독립 가정을 비판적으로 분석합니다.

-SME 도메인 결합: 산업 및 시스템공학 관점에서 데이터 기반 의사결정 및 최적화 기법을 인공지능 모델에 투영합니다.
